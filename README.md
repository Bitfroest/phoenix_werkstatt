# phoenix_werkstatt
shows online person in phoenix workshop
http://werkstatt.tum-phoenix.de/

## update.php API
### GET
  - persons
    - all information about all persons in JSON 
  - person
    - set id as value
    - all information about a person by 'id' in JSON
  - add
  - time
  - reset
    
### POST
  - ?add
    - firstname
    - lastname
  - ?time
    - id
    - minutes
  - ?reset
    - id
